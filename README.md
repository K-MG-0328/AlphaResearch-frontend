# AlphaResearch Frontend

Next.js 16 · React 19 · Jotai · Tailwind CSS 기반 투자 정보 플랫폼 프런트엔드.

## Getting Started

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) 에서 확인.

## Tech

- Next.js 16 (App Router), React 19, TypeScript 5 (strict)
- Tailwind CSS 4, Jotai (전역 상태), ESLint 9
- 폰트: [Geist](https://vercel.com/font) via `next/font`
